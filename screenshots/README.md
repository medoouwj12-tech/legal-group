# Screenshots

Drop your screen captures here using the following naming convention so they can be referenced directly from `README.md`.

## Suggested Files

| File name                       | What to capture                                         | Recommended size |
| ------------------------------- | ------------------------------------------------------- | ---------------- |
| `home-desktop-en.png`           | Homepage in English, full screen (1440×900)             | 1440 × 900+      |
| `home-desktop-ar.png`           | Homepage after switching to Arabic (RTL layout proves)  | 1440 × 900+      |
| `home-mobile-en.png`            | Homepage on mobile (375 × 812)                          | 375 × 812        |
| `services-desktop-en.png`       | Services page to show the 6-card grid                   | 1440 × 900+      |
| `contact-desktop-en.png`        | Contact form + map area                                 | 1440 × 900+      |
| `counter-animation.gif` *(opt)* | Animated counter in motion — sells the polish factor    | 600 × 400        |

## Capture Tips (if you use the `matrix` MCP tool or your browser)

- **HiDPI**: capture at 2× for crisp GitHub rendering
- **Mobile viewports**: use Chrome DevTools device toolbar, choose iPhone 12/13/14
- **Light mode is fine** — this design is always dark
- **Switch language by appending `?lang=ar` to the URL**

## Linking From README

Once uploaded, reference them in `README.md` like:

```markdown
![Homepage EN](./screenshots/home-desktop-en.png)
![Homepage AR (RTL)](./screenshots/home-desktop-ar.png)
```
